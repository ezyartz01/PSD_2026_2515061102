Program ini merupakan simulasi fitur Add Friend pada sebuah game yang menggunakan algoritma Binary Search untuk mencari Player ID secara cepat dan efisien. Daftar pemain online dibuat dalam keadaan terurut dari ID 1001 sampai 1020 agar proses Binary Search dapat berjalan dengan benar.

Fungsi binary_search_player() digunakan untuk mencari posisi Player ID berdasarkan batas kiri (l) dan batas kanan (r) pencarian, kemudian menentukan nilai tengah untuk dibandingkan dengan ID yang dicari. Jika ditemukan, fungsi mengembalikan indeks pemain, sedangkan jika tidak ditemukan akan mengembalikan nilai -1.

Pada fungsi main(), program menampilkan friend list yang dimiliki pemain, kemudian menanyakan apakah pengguna ingin menambah teman baru. Jika pengguna memilih menambah teman, sistem akan mencari Player ID menggunakan Binary Search, melakukan validasi agar tidak terjadi duplikasi teman, serta meminta konfirmasi sebelum menambahkan pemain ke dalam friend list. Program berjalan berulang hingga pengguna memilih keluar, lalu menampilkan daftar teman akhir.
